# Portafolio Audiovisual — Guía para Cliente

## Qué es
Un sitio web para mostrar su trabajo fotográfico y audiovisual con una estética elegante y minimalista.

## Cómo se usa
- Precarga: toque o haga clic para entrar. Al mover el dedo o el mouse sobre la cámara, esta se inclina suavemente.
- Menú burbuja: lo acompaña mientras navega, puede abrirse/cerrarse y moverse arrastrándolo.
- Botón “Sonido”: reproduce el sonido de apertura de cámara para probar.
- Botón “Tema”: cambia entre modo claro y oscuro. El sitio recordará su preferencia.

## Secciones
- Acerca de mí: presentación y propuesta.
- Fotos: selección de trabajos y proyectos.
- Contacto: formulario para enviar un mensaje.
- Reseñas: testimonios de clientes.

## Experiencia en móviles
- La burbuja se arrastra fácilmente con el dedo y no interfiere con el desplazamiento.
- La cámara de la precarga responde al movimiento del dedo y vuelve a su posición al soltar.

## Estilo visual
- Fuentes: títulos con Bodoni Moda (elegante y editorial) y texto con Manrope (limpia y moderna).
- Paleta: colores suaves con acento en violeta para resaltar acciones.

## Personalización
- Textos, fotos, colores y secciones pueden ajustarse a su preferencia.
- Si desea agregar nuevas páginas o vínculos (por ejemplo, a redes sociales), se incorporan sin complicaciones.

## Exclusividad
Este proyecto es exclusivo para usted y no está destinado a uso por terceros. Cualquier difusión o reutilización se realizará solo bajo su solicitud.

## Soporte
Para cambios, mejoras o nuevas ideas, contácteme y coordinamos los ajustes.
